- **개요:** 차량 상태 데이터(RPM, 속도, 기어, GPS 등)와 영상 정보를 동기화하여 저장  
- **기술:** Raspberry Pi 5, C/Python 하이브리드, Hailo-8 AI 추론, GStreamer 스트리밍  
- **성과:** 사고 전후(-10s/+10s) 영상 저장 및 위험 감지 기반 이벤트 기록 시스템 구현
